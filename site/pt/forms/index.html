    Coleta de Dados

    A área de Coleta de Dados é o centro para todas as entradas de dados baseadas em formulários no Dino. A partir daqui, você pode acessar todos os esquemas de formulário com os quais sua equipe trabalha, enviar novas entradas, revisar envios existentes e explorar os dados coletados em um mapa ou por meio de análises assistidas por IA.

    Quando você abre a área de Coleta de Dados, verá uma lista de todos os esquemas de formulário disponíveis. Cada item na lista mostra o nome e o ícone do formulário.

    Navegando na Área

    A página principal mostra todos os esquemas de formulário aos quais você tem acesso. Você pode interagir com esta lista de várias maneiras:

    1. Abrir um Formulário: Clique em qualquer esquema de formulário na lista para abrir sua lista de envios. É aqui que você pode navegar, adicionar ou gerenciar entradas individuais do formulário.
    2. Pesquisar e Filtrar: Use a barra de pesquisa no topo para encontrar rapidamente um formulário específico pelo seu nome.
    3. Ordenar: Clique nos cabeçalhos das colunas (como "Nome") para ordenar a lista.

    Acesso Rápido

    A lista de formulários é seu ponto de partida para todas as atividades de coleta de dados. Pense em cada esquema de formulário como um tipo diferente de relatório ou pesquisa que você precisa preencher.

    Trabalhando com um Esquema de Formulário

    Após selecionar um esquema de formulário, você é levado à sua página dedicada. Aqui, você pode realizar várias ações principais:

    1. Ver Envios: Veja uma tabela com todos os dados previamente enviados para este formulário.
    2. Criar um Novo Envio: Clique no botão Criar para preencher e enviar um novo formulário.
    3. Explorar Dados:
      • Clique na aba Mapa para ver todos os envios plotados geograficamente.
      • Clique na aba DataChat para fazer perguntas sobre seus dados coletados usando IA.

    Fluxos de Trabalho Principais

    Para enviar novos dados:

    1. Na página principal de Coleta de Dados, clique no esquema de formulário desejado.
    2. Na página da lista de envios do formulário, clique no botão Criar.
    3. Preencha todos os campos obrigatórios no formulário que se abre.
    4. Clique em Enviar para salvar sua entrada.

    Para revisar dados existentes:

    1. Na página principal de Coleta de Dados, clique no esquema de formulário relevante.
    2. Navegue pela lista de envios. Você pode clicar em qualquer entrada individual para Visualizar seus detalhes ou Editar se tiver permissão.

    Gerenciando Esquemas de Formulário

    Criar e editar a estrutura dos esquemas de formulário (adicionar campos, alterar perguntas) requer permissões adicionais. Se você precisar configurar um novo formulário ou modificar um existente, consulte Gerenciando Esquemas de Formulário ou entre em contato com seu administrador.

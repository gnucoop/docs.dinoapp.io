    Preencher um Formulário

    Esta página explica como preencher um novo envio de formulário. Você pode iniciar um novo envio selecionando um formulário da lista e clicando no botão +.

    Selecionando Métricas

    Etapa Opcional

    Esta etapa só aparece se o seu formulário usar métricas contextuais (como projetos ou locais). Se você vir os campos do formulário diretamente, pule para a próxima seção.

    O primeiro passo é selecionar o contexto para o seu envio. Isso vincula seus dados a projetos, locais ou outras categorias específicas.

    1. Use os menus suspensos para escolher um valor para cada métrica obrigatória.
    2. Clique em Ir para o formulário ou na seta para frente para prosseguir para os campos de entrada de dados.

    Você pode retornar a esta etapa a qualquer momento clicando na seta para trás na parte superior da página.

    Entrada de Áudio

    Se um botão de microfone estiver visível, você pode gravar uma resposta falada. O Dino tentará transcrevê-la e preencher os campos relevantes automaticamente.

    Preenchendo os Campos do Formulário

    Insira seus dados nos campos da página. Os campos marcados como obrigatórios devem ser preenchidos antes que você possa salvar o envio final.

    Se o formulário estiver dividido em várias seções, você pode alternar entre elas usando as abas ou a barra de navegação perto do topo do formulário.

    Uma barra de status pode mostrar o estágio atual do fluxo de trabalho do seu envio.

    Salvando Seu Envio

    Depois de inserir seus dados, use os botões no canto inferior direito para salvar:

    • Salvar formulário (ícone de disquete): Salva o envio como final. Este botão só fica ativo quando todos os campos obrigatórios estão preenchidos corretamente.
    • Salvar rascunho (ícone de salvar como): Salva seu progresso sem concluir toda a validação. Use isso para finalizar seu envio mais tarde.

    Uploads de Arquivos Offline

    Se o seu formulário incluir anexos de arquivo e você estiver trabalhando offline, um aviso aparecerá ao salvar. Os dados do envio serão salvos, mas os arquivos anexados ficarão na fila para upload assim que você estiver online novamente.

    Uma mensagem de confirmação aparecerá quando seu envio for salvo com sucesso.

    Discuter avec vos données (IA)

    La page de discussion de données vous permet de poser des questions en langage simple sur les soumissions collectées pour un formulaire. L'assistant IA analyse les données et renvoie des réponses sous forme de texte, de tableaux ou de graphiques.

    Fonctionnalité optionnelle

    Cette fonctionnalité nécessite une clé API DataChat et peut ne pas être disponible dans votre installation. Si vous ne voyez pas cette option, contactez votre administrateur.

    Accepter les conditions d'utilisation

    La première fois que vous utilisez la discussion de données, une déclaration des conditions d'utilisation vous sera présentée. Lisez-la et cliquez sur Accepter pour continuer.

    Votre acceptation est enregistrée sur votre appareil, vous ne serez donc pas redemandé lors de vos prochaines visites.

    Saisir votre clé API

    Après avoir accepté les conditions, il vous sera demandé de saisir votre clé API DataChat.

    1. Tapez votre clé API dans le champ.
    2. Appuyez sur Entrée ou cliquez sur le bouton fléché pour la soumettre.

    Le système vérifiera la clé. Si elle est valide, l'interface de discussion se chargera. Votre clé est enregistrée sur votre appareil, vous n'avez donc pas besoin de la ressaisir la prochaine fois.

    Clé API invalide

    Si la clé est rejetée, vérifiez que vous l'avez saisie correctement. Un message d'erreur apparaîtra sous le champ si la clé n'est pas reconnue.

    Service indisponible

    Si le message "DINO-AI ne répond pas pour le moment" apparaît, le service peut être temporairement indisponible. Réessayez plus tard.

    Poser des questions

    Une fois la discussion prête, vous verrez un aperçu de vos données et un champ de saisie de texte en bas de l'écran.

    1. Cliquez à l'intérieur du champ de texte et tapez votre question (par exemple : "Combien d'entrées ont été soumises le mois dernier ?" ou "Montrez-moi un graphique des réponses par emplacement").
    2. Appuyez sur Entrée pour envoyer.
    3. L'assistant traitera votre question et affichera la réponse dans l'historique de discussion ci-dessus. Les réponses peuvent apparaître sous forme de texte, de tableaux ou de graphiques.

    Vous pouvez poursuivre la conversation en posant des questions complémentaires. Si l'assistant suggère des questions de suivi, vous pouvez cliquer dessus pour les envoyer directement.

    Donner votre avis

    Chaque réponse comprend un bouton pouce vers le haut et pouce vers le bas. Utilisez-les pour évaluer la qualité de la réponse. Vos retours aident à améliorer l'assistant au fil du temps.

    Dépannage

    "Aucun formulaire n'a été trouvé"

    Warning

    Aucune soumission n'est encore associée à ce formulaire. Ajoutez quelques entrées en utilisant la liste des soumissions avant d'utiliser la discussion de données.

    "Désolé, vous n'avez pas assez de crédits !"

    Warning

    Votre compte a épuisé ses crédits DataChat. Un lien peut apparaître pour acheter des crédits supplémentaires. Si vous avez besoin d'aide, contactez votre administrateur.

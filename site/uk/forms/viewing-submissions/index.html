    Перегляд та керування надісланими даними

    На цій сторінці відображаються всі надіслані дані для конкретної схеми форми. Ви можете переглядати, шукати, дивитися деталі, редагувати, експортувати та керувати окремими записами з цього списку.

    Перегляд списку

    Список відображає всі надіслані дані для обраної форми. Загальна кількість записів, що відповідають вашим поточним фільтрам, показана вгорі.

    Використовуйте елементи керування пагінацією, щоб переходити між сторінками, або перейти на першу чи останню сторінку.

    Ви також можете змінити, які стовпці відображаються, натиснувши кнопку вибору стовпців (значок сітки) у заголовку таблиці.

    Пошук та фільтрація

    Панель фільтрів над списком дозволяє звузити, які записи відображаються:

    • Пошук за ключовим словом — введіть будь-яке слово, щоб відфільтрувати записи, поля яких містять цей текст.
    • Дата з / Дата по — показувати лише записи, створені в межах діапазону дат.
    • Фільтри за метриками — фільтрувати за проектом, місцем, районом, справою, організацією або іншими пов'язаними даними.
    • Фільтр за статусом — показувати лише записи з певним статусом робочого процесу.
    • Фільтр за користувачем — показувати лише записи, створені конкретним користувачем або групою користувачів.
    • Збережені фільтри (пресети) — збережіть поточну комбінацію фільтрів як пресет для швидкого доступу пізніше. Використовуйте меню пресетів у панелі фільтрів, щоб зберегти, завантажити або видалити пресети.

    Щоб видалити фільтр, натисніть × всередині цього поля.

    Дії з рядком

    Кожен рядок має набір значків дій праворуч. Доступні дії залежать від ваших дозволів:

    • Перегляд (значок ока) — відкрити запис у режимі лише для читання.
    • Редагувати (значок олівця) — відкрити запис для редагування.
    • Друк (значок принтера) — створити PDF-версію запису.
    • Експортувати як Word (значок документа) — завантажити запис як документ Word.
    • Дублювати (значок копіювання) — створити новий чернетковий запис, попередньо заповнений даними цього запису.
    • Видалити (значок кошика) — остаточно видалити запис. З'явиться підтвердження.
    • Друк бейджа (значок бейджа) — створити документ бейджа для цього запису (доступно, якщо форма прив'язана до справи).
    • Переглянути журнал (значок історії) — переглянути історію змін, внесених до цього запису.

    Дії на основі дозволів

    Не всі дії видимі кожному користувачеві. Показані дії залежать від призначених вам дозволів. Деякі дії, такі як редагування або видалення, можуть бути доступні лише якщо статус запису це дозволяє.

    Масові дії

    Щоб діяти з кількома записами одночасно:

    1. Виберіть записи, якими потрібно керувати, поставивши прапорець у їхніх рядках.
    2. Використовуйте кнопки масових дій, що з'являються на панелі інструментів, щодо видалити або редагувати всі вибрані записи одночасно.

    Обмеження масового редагування

    Масове редагування доступне лише якщо всі вибрані записи мають статус, що дозволяє редагування.

    Додавання нового запису

    Натисніть кнопку + (плаваюча кругла кнопка внизу праворуч на сторінці), щоб відкрити порожню форму та записати нові надіслані дані.

    Імпорт записів

    Натисніть кнопку імпорту (значок завантаження з хмари, плаваюча кнопка внизу праворуч), щоб масово імпортувати записи з файлу.

    Експорт списку

    Натисніть кнопку Експорт (значок завантаження) на панелі фільтрів, щоб завантажити поточний список записів як файл.

    Перегляд даних на карті

    Якщо надіслані дані містять географічну інформацію, натисніть кнопку Карта на панелі фільтрів, щоб відкрити Перегляд на карті для цієї форми.

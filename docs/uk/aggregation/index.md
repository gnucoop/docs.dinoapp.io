# Агрегація

Сторінка Агрегації надає уніфікований перегляд усіх записів форм у вашій організації. Замість перегляду записів для однієї схеми форми за раз, ви можете бачити записи з кількох схем разом у єдиному списку з можливістю фільтрації.

![Агрегація](../imgs/aggregation/index.png)

## Перегляд списку

Список відображає всі записи, які ви маєте дозвіл переглядати. Кожен рядок представляє один запис. За замовчуванням відображаються такі стовпці:

*   **Схема форми**: Назва схеми форми, до якої належить цей запис.
*   **Статус**: Поточний статус робочого процесу запису.
*   Також можуть з'являтися додаткові стовпці для метрик, таких як **Проєкт**, **Місцезнаходження** або **Організація**, залежно від конфігурації вашої системи.

Ви можете натиснути на будь-який рядок, щоб вибрати його або розгорнути для перегляду детальнішої інформації.

## Фільтрація списку

Над списком розташований панель фільтрів. Використовуйте його, щоб звузити перелік записів. Ви можете фільтрувати за:

*   Проєктом
*   Місцезнаходженням
*   Областю
*   Справою
*   Кодом справи
*   Організацією
*   Статусом форми
*   Користувачем

!!! tip "Поради щодо фільтрації"
    Панель фільтрів Агрегації призначена для швидкої фільтрації між схемами. Для розширених функцій, таких як збережені набори фільтрів або експорт даних, перейдіть до списку записів для конкретної схеми форми.

## Дії з рядками

Коли ви наводите курсор на рядок, праворуч з'являється набір значків дій. Дії, доступні для конкретного запису, залежать від ваших дозволів для його схеми форми.

*   **Перегляд** (![значок ока]()): Відкрити запис у режимі лише для читання.
*   **Редагувати** (![значок олівця]()): Відкрити запис для редагування.
*   **Друк** (![значок принтера]()): Створити та відкрити PDF-версію запису. Вас попросять підтвердити перед створенням PDF.
*   **Видалити** (![значок видалення]()): Остаточно видалити запис. Вас попросять підтвердити цю дію.

## Створення нового запису

Ви можете розпочати новий запис форми безпосередньо зі сторінки Агрегації.

1.  Натисніть кнопку **+** (Додати) у правому нижньому куті екрана.
2.  Відкриється діалогове вікно зі списком схем форм, для яких ви маєте дозвіл створювати записи.
3.  Виберіть потрібну схему форми зі списку.
4.  Натисніть **Створити форму**. Ви перейдете до форми, щоб почати її заповнення.

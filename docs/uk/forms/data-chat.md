# Спілкування з вашими даними (AI)

Сторінка Data Chat дозволяє задавати питання простою мовою про відправлені дані для форми. AI-асистент аналізує дані та повертає відповіді у вигляді тексту, таблиць або діаграм.

![Спілкування з вашими даними (AI)](../../imgs/forms/data-chat.png)

!!! note "Опціональна функція"
    Ця функція вимагає ключ API DataChat і може бути недоступною у вашій установці. Якщо ви не бачите цієї опції, зверніться до вашого адміністратора.

---

## Прийняття Умов використання

Під час першого використання Data Chat вам буде показано заяву про умови використання. Прочитайте її та натисніть **Прийняти**, щоб продовжити.

Ваше прийняття зберігається на вашому пристрої, тому при наступних відвідуваннях вас більше не запитуватимуть.

---

## Введення вашого ключа API

Після прийняття умов вас попросять ввести ваш **ключ API DataChat**.

1. Введіть ваш ключ API у поле.
2. Натисніть **Enter** або клацніть кнопку **зі стрілкою**, щоб відправити його.

Система перевірить ключ. Якщо він дійсний, завантажиться інтерфейс чату. Ваш ключ зберігається на пристрої, тому вам не потрібно вводити його знову наступного разу.

!!! warning "Недійсний ключ API"
    Якщо ключ відхилено, перевірте, чи ви ввели його правильно. Під полем з'явиться повідомлення про помилку, якщо ключ не розпізнано.

!!! warning "Сервіс недоступний"
    Якщо з'явиться повідомлення *"DINO-AI зараз не відповідає"*, сервіс може бути тимчасово недоступним. Спробуйте пізніше.

---

## Задавання питань

Коли чат готовий, ви побачите попередній перегляд ваших даних та текстове поле введення внизу екрана.

1. Клацніть усередині текстового поля та введіть ваше питання (наприклад: *"Скільки записів було відправлено минулого місяця?"* або *"Покажіть мені діаграму відповідей за місцем розташування"*).
2. Натисніть **Enter**, щоб відправити.
3. Асистент обробить ваше питання та відобразить відповідь в історії чату вище. Відповіді можуть з'являтися у вигляді тексту, таблиць або діаграм.

Ви можете продовжити розмову, задаючи уточнюючі питання. Якщо асистент запропонує уточнюючі питання, ви можете клацнути на них, щоб відправити безпосередньо.

---

## Надання відгуку

Кожна відповідь містить кнопки "палець вгору" та "палець вниз". Використовуйте їх, щоб оцінити якість відповіді. Ваш відгук допомагає покращувати асистента з часом.

---

## Вирішення проблем

### "Форми не знайдено"

!!! warning
    З цією формою ще не пов'язано жодних відправлених даних. Додайте кілька записів за допомогою списку відправлених даних перед використанням Data Chat.

### "На жаль, у вас недостатньо кредитів!"

!!! warning
    На вашому обліковому записі закінчилися кредити DataChat. Може з'явитися посилання для придбання додаткових кредитів. Якщо вам потрібна допомога, зверніться до вашого адміністратора.

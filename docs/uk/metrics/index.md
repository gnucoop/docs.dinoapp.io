# Метрики

Метрики — це категорії довідкових даних, які використовуються в Dino для класифікації, організації та фільтрації зібраних даних. Під час заповнення форми або створення звіту ви обираєте значення з цих категорій — наприклад, вибираєте проект, місцезнаходження або організацію.

Розділ «Метрики» — це місце, де ви керуєте списками доступних значень для кожної категорії. Він служить центральним вузлом для всіх ваших довідкових даних.

## Типи метрик

На головній сторінці відображаються типи метрик, активні у вашій установці Dino. Кожен тип показаний у вигляді картки з іконкою та міткою. Натисніть на будь-яку картку, щоб відкрити її сторінку керування.

Залежно від конфігурації вашої системи, можуть бути доступні деякі або всі наступні типи метрик:

| Тип метрики | Опис |
|---|---|
| **Тематичні напрямки** | Сфери роботи або тематичні групи для вашої діяльності. |
| **Справи** | Окремі справи, люди або бенефіціари, за якими відстежуються дані в заповнених формах. |
| **Місцезнаходження** | Географічні місця, де збираються дані або відбувається діяльність. |
| **Проекти** | Проекти, до яких прив'язуються заповнені форми та звіти. |
| **Організації** | Організації, залучені до діяльності або відповідальні за неї. |

!!! tip "Доступ до метрик"
    Ви можете перейти до розділу «Метрики», натиснувши **Метрики** у головному меню програми.

---

## Що можна робити

На головній сторінці «Метрики» ви можете:

1.  **Переглянути всі активні типи метрик**, доступні для ваших даних.
2.  **Перейти до конкретного типу метрики**, натиснувши на його картку. Це перенесе вас на спеціальну сторінку, де ви можете керувати списком значень для цього типу (наприклад, додати нове місцезнаходження або відредагувати назву проекту).
3.  **Використовувати навігаційний ланцюжок (breadcrumbs)** у верхній частині сторінки, щоб відстежувати свій шлях навігації в розділі «Метрики».

Детальні інструкції щодо додавання, редагування або видалення значень у межах конкретного типу метрики дивіться в розділі [Керування метриками](managing-metrics.md).

# Створення звіту

Щоб створити новий звіт, перейдіть до розділу "Звіти", відкрийте схему звіту та натисніть кнопку **+** (плаваюча кругла кнопка в правому нижньому куті сторінки).

![Створення звіту](../../imgs/reports/creating-a-report.png)

!!! note "Потрібен дозвіл"
    Кнопка додавання видима лише якщо у вас є дозвіл на створення звітів для цієї схеми.

---

## Крок 1 — Метрики звіту

Якщо схема звіту вимагає метрик, на першому кроці вам потрібно вибрати значення метрик, які будуть включені в цей звіт (наприклад: проект, локація або організація).

1.  Заповніть усі обов'язкові поля для метрик.
2.  Натисніть **Далі**, щоб продовжити.

---

## Крок 2 — Дані звіту

На другому кроці заповніть наступні поля:

- **Назва звіту** *(обов'язково)* — Назва для ідентифікації цього звіту.
- **Дані з** *(необов'язково)* — Початок діапазону дат для даних, що включаються в звіт.
- **Дані до** *(необов'язково)* — Кінець діапазону дат.

Діапазон дат є необов'язковим. Якщо залишити його порожнім, звіт включатиме всі доступні дані для вибраних метрик.

!!! tip "Схеми без метрик"
    Якщо схема звіту не використовує метрики, ви побачите лише поля для назви звіту та діапазону дат без кроків.

---

## Збереження звіту

Натисніть кнопку **Зберегти звіт** (плаваюча кругла кнопка), щоб згенерувати та зберегти звіт.

- Якщо звіт успішно згенеровано, з'явиться підтвердження, і вас буде повернуто до списку звітів.

!!! warning "Звіти на основі ШІ"
    Деякі схеми звітів використовують ШІ для генерації вмісту. Створення звіту за такими схемами споживає кредити ШІ. Вартість у кредитах показана у спливаючій підказці на кнопці додавання перед початком. Якщо на вашому акаунті недостатньо кредитів, з'явиться повідомлення, і створити звіт буде неможливо.

# Керування мовами

Сторінка **Мови** дозволяє адміністраторам керувати всім перекладеним текстом, який використовується в Dino. Звідси ви можете переглядати, редагувати та додавати переклади, керувати доступними мовами та експортувати файли перекладів для резервного копіювання або редагування.

![Керування мовами](../imgs/administration/languages.png)

!!! warning "Тільки для адміністраторів"
    Ця область видима лише користувачам з роллю Адміністратора. Якщо ви не бачите її в навігації, зверніться до системного адміністратора.

## Перегляд перекладів

Основний вигляд показує список усіх записів перекладів. Кожен запис відображає свій **ключ** — внутрішній ідентифікатор, який використовує програма — і, коли мова вибрана, відповідний перекладений текст.

Під час завантаження даних перекладу відображається індикатор завантаження.

### Фільтрація списку

Два елементи керування вгорі сторінки дозволяють звузити показані записи:

- **Пошук за ключовим словом** — введіть будь-яке слово, щоб відфільтрувати записи, ключ або переклад яких містить цей текст. Список оновлюється під час введення.
- **Вибір мови** — ряд кнопок показує **Ключ** та одну кнопку для кожної доступної мови. Натисніть назву мови, щоб відобразити переклади цієї мови поруч із кожним ключем. Записи без перекладу для обраної мови відображаються як *(Немає перекладу)*.

---

## Редагування запису перекладу

1. Натисніть на будь-який запис у списку, щоб відкрити діалогове вікно **Редагувати переклад**.
2. У діалоговому вікні відображається **ключ** та текстове поле для кожної доступної мови.
3. Оновіть переклади за потребою.
4. Натисніть **Зберегти**, щоб застосувати зміни, або **Скасувати**, щоб закрити без збереження.

Ви також можете остаточно видалити окремий запис із цього діалогового вікна, натиснувши кнопку **Видалити**. Це видалить ключ перекладу та всі пов'язані з ним переклади.

!!! warning
    Видалення запису перекладу є остаточним. Ключ і всі його значення для різних мов будуть видалені.

## Додавання нового запису перекладу

Використовуйте це, коли вам потрібно додати ключ перекладу, якого ще немає в системі.

1. Натисніть кнопку **+ Переклад** на панелі інструментів.
2. Відкриється діалогове вікно **Додати переклад**. Воно містить одне текстове поле для кожної поточної активної мови.
3. Введіть текст перекладу для кожної мови за потребою.
4. Натисніть **Зберегти**, щоб додати новий запис, або **Скасувати**, щоб відмінити.

Після збереження запису на короткий час з'явиться підтвердження.

## Керування мовами

Використовуйте це, щоб додати нову мову, оновити переклади існуючої мови або видалити власний набір перекладів.

1. Натисніть кнопку **Мова** на панелі інструментів.
2. Відкриється діалогове вікно **Налаштування мов**. Воно показує список доступних мов і надає такі дії:

### Додавання нової мови

1. Натисніть кнопку **+** у верхній частині діалогового вікна.
2. З'явиться форма з проханням вказати **назву мови** (назва, яка з'явиться в інтерфейсі, наприклад "French" або "fr").
3. За бажанням завантажте **JSON-файл перекладу**, натиснувши **Додати JSON** і вибравши файл з пристрою. Вміст файлу буде попередньо переглянуто перед збереженням.
4. Натисніть **Зберегти**, щоб додати мову, або **Скасувати**, щоб відмінити.

### Перегляд існуючої мови

Натисніть кнопку з назвою мови, щоб вибрати її. У діалоговому вікні буде показано попередній перегляд усіх ключів і значень перекладів, що наразі зберігаються для цієї мови.

### Оновлення перекладів мови

З вибраною мовою натисніть **Оновити переклад**, щоб завантажити новий JSON-файл. Діалогове вікно попередньо покаже зміни — додані та змінені ключі — перед збереженням.

1. Натисніть **Оновити переклад** і виберіть JSON-файл з пристрою.
2. Перегляньте попередній перегляд, який показує додані та змінені рядки.
3. Натисніть **Зберегти**, щоб застосувати оновлення, або **Скасувати**, щоб відмінити.

### Видалення власного перекладу

З вибраною мовою натисніть **Видалити власний переклад**, щоб видалити власні дані перекладу для цієї мови.

!!! warning
    Це видаляє власні переклади для обраної мови. Сама мова може залишитися в системі, але її власний вміст буде втрачено.

## Експорт перекладів

Ви можете завантажити дані перекладу для будь-якої мови у вигляді JSON-файлу.

1. Натисніть кнопку **Експорт** (значок завантаження) на панелі інструментів.
2. Відкриється діалогове вікно **Експорт**, яке показує список доступних мов.
3. Натисніть назву мови, яку хочете експортувати. Праворуч з'явиться попередній перегляд її даних перекладу.
4. Натисніть **Завантажити**, щоб зберегти файл на пристрій.

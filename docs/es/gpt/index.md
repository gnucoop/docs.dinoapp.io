# DinoGPT

DinoGPT es un asistente de IA que te permite hacer preguntas en lenguaje natural y recibir respuestas extraídas de una base de conocimiento configurada. Está diseñado para conversaciones directas: escribes una pregunta y recibes una respuesta, junto con referencias a los documentos de origen en los que se basa la respuesta.

![DinoGPT](../../imgs/gpt/index.png)

!!! note "Función opcional"
    Esta función requiere una clave API de DINO-AI y puede no estar disponible en tu instalación. Si no ves esta opción, contacta con tu administrador.

---

## Aceptar los Términos de Uso

La primera vez que abras DinoGPT, se te mostrará una declaración de términos de uso que explica cómo funciona el asistente, qué puede y no puede hacer, y tus responsabilidades como usuario. Lee los términos y haz clic en **Aceptar** para continuar.

Tu aceptación se guarda en tu dispositivo, por lo que no se te volverá a preguntar en visitas futuras.

---

## Introducir tu Clave API

Después de aceptar los términos, se te pedirá que introduzcas tu **clave API de DINO-AI**.

1. Escribe tu clave API en el campo.
2. Pulsa **Intro** o haz clic en el **botón de flecha** para enviarla.

El sistema verificará la clave. Si es válida, aparecerá un mensaje de confirmación y se cargará la interfaz de chat. Tu clave se guarda en tu dispositivo, por lo que no necesitarás volver a introducirla la próxima vez.

!!! warning "Clave API no válida"
    Si la clave es rechazada, verifica que la has introducido correctamente. Aparecerá un mensaje de error debajo del campo si la clave no es reconocida.

!!! warning "Servicio no disponible"
    Si aparece el mensaje *"DINO-AI no responde en este momento"*, el servicio puede estar temporalmente no disponible. Inténtalo de nuevo más tarde.

---

## Hacer Preguntas

Una vez que el chat esté listo, el asistente te saludará y aparecerá un campo de texto en la parte inferior de la pantalla.

1. Selecciona un **Espacio de nombres** en el menú desplegable para elegir de qué base de conocimiento debe extraer información el asistente.
2. Haz clic dentro del campo de texto y escribe tu pregunta.
3. Pulsa **Intro** para enviarla.
4. El asistente procesará tu pregunta y mostrará la respuesta en el historial de chat de arriba.

Puedes continuar la conversación haciendo preguntas de seguimiento. El asistente tiene en cuenta todo el historial de la conversación al generar cada respuesta.

---

## Comprender las Respuestas

Cada respuesta del asistente puede incluir:

- **Texto de la respuesta** — la respuesta del asistente a tu pregunta.
- **Fuentes** — fichas clicables que muestran el documento o archivo del que se extrajo la respuesta, junto con el número de página si está disponible. Haz clic en una ficha de fuente para leer el pasaje relevante.
- **Preguntas sugeridas** — fichas clicables con preguntas de seguimiento sugeridas por el asistente. Haz clic en una para enviarla directamente.

---

## Proporcionar Retroalimentación

Las respuestas incluyen un botón de **pulgar arriba** y **pulgar abajo**. Utilízalos para calificar la calidad de la respuesta. Aparecerá un mensaje de confirmación cuando se envíe tu retroalimentación.

---

## Resolución de Problemas

### El chat ha alcanzado su longitud máxima

!!! warning
    Si el asistente responde que el chat ha alcanzado su longitud máxima, recarga la página para iniciar una nueva conversación.

### "¡Lo siento, no tienes suficientes créditos!"

!!! warning
    Tu cuenta se ha quedado sin créditos de DINO-AI. Puede aparecer un enlace para comprar créditos adicionales. Si necesitas ayuda, contacta con tu administrador.

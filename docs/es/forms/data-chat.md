# Chatear con tus datos (IA)

La página de Chat de Datos te permite hacer preguntas en lenguaje natural sobre los envíos recopilados para un formulario. El asistente de IA analiza los datos y devuelve respuestas en texto, tablas o gráficos.

![Chatear con tus datos (IA)](../imgs/forms/data-chat.png)

!!! note "Función opcional"
    Esta función requiere una clave API de DataChat y puede no estar disponible en tu instalación. Si no ves esta opción, contacta con tu administrador.

---

## Aceptar los Términos de Uso

La primera vez que uses Chat de Datos, se te mostrará una declaración de términos de uso. Léela y haz clic en **Aceptar** para continuar.

Tu aceptación se guarda en tu dispositivo, por lo que no se te volverá a preguntar en visitas futuras.

---

## Introducir tu Clave API

Después de aceptar los términos, se te pedirá que introduzcas tu **clave API de DataChat**.

1. Escribe tu clave API en el campo.
2. Pulsa **Intro** o haz clic en el **botón de flecha** para enviarla.

El sistema verificará la clave. Si es válida, se cargará la interfaz de chat. Tu clave se guarda en tu dispositivo para que no tengas que volver a introducirla la próxima vez.

!!! warning "Clave API no válida"
    Si la clave es rechazada, comprueba que la has introducido correctamente. Aparecerá un mensaje de error debajo del campo si la clave no es reconocida.

!!! warning "Servicio no disponible"
    Si aparece el mensaje *"DINO-AI no responde en este momento"*, el servicio puede estar temporalmente no disponible. Inténtalo de nuevo más tarde.

---

## Hacer Preguntas

Una vez que el chat esté listo, verás una vista previa de tus datos y un campo de texto en la parte inferior de la pantalla.

1. Haz clic dentro del campo de texto y escribe tu pregunta (por ejemplo: *"¿Cuántas entradas se enviaron el mes pasado?"* o *"Muéstrame un gráfico de respuestas por ubicación"*).
2. Pulsa **Intro** para enviar.
3. El asistente procesará tu pregunta y mostrará la respuesta en el historial de chat de arriba. Las respuestas pueden aparecer como texto, tablas o gráficos.

Puedes continuar la conversación haciendo preguntas de seguimiento. Si el asistente sugiere preguntas de seguimiento, puedes hacer clic en ellas para enviarlas directamente.

---

## Proporcionar Retroalimentación

Cada respuesta incluye un botón de pulgar hacia arriba y pulgar hacia abajo. Úsalos para calificar la calidad de la respuesta. Tu retroalimentación ayuda a mejorar el asistente con el tiempo.

---

## Resolución de Problemas

### "No se encontraron formularios"

!!! warning
    Aún no hay envíos asociados con este formulario. Añade algunas entradas usando la lista de envíos antes de usar Chat de Datos.

### "¡Lo siento, no tienes suficientes créditos!"

!!! warning
    Tu cuenta se ha quedado sin créditos de DataChat. Puede aparecer un enlace para comprar créditos adicionales. Si necesitas ayuda, contacta con tu administrador.

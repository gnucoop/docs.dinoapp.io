# Panel de control

El Panel de control es la primera pantalla que ves después de iniciar sesión. Sirve como tu centro de operaciones para navegar por las funciones de Dino. Dependiendo de la configuración de tu sistema, aparecerá en uno de dos diseños: un **Panel de control de menú** o un **Panel de control de informes**.

![Panel de control](../../imgs/dashboard/index.png)

---

## Panel de control de menú

En este diseño, el Panel de control presenta una cuadrícula de tarjetas de navegación. Cada tarjeta proporciona acceso rápido a un área principal de la aplicación para la que tienes permiso de uso.

Normalmente verás las siguientes tarjetas:

*   **Formularios**: Navega al área de [Formularios](../forms/index.md) para crear nuevos esquemas de formularios, recopilar datos y revisar envíos.
*   **Informes**: Navega al área de [Informes](../reports/index.md) para crear, ver y gestionar informes basados en tus datos recopilados.
*   **Métricas**: Navega al área de [Métricas](../metrics/index.md) para gestionar datos de referencia como proyectos, ubicaciones y organizaciones.
    !!! warning "Visibilidad"
        La tarjeta de Métricas está oculta si tu cuenta de usuario tiene permisos de solo invitado.
*   **Usuarios**: Navega al área de [Usuarios](../users/index.md) para gestionar cuentas de usuario y grupos.
    !!! tip "Acceso de administrador"
        La tarjeta de Usuarios solo es visible para usuarios con privilegios de administrador.

Para navegar, simplemente haz clic en la tarjeta del área a la que deseas acceder.

## Panel de control de informes

En este diseño, tu Panel de control está personalizado para mostrar un único informe que hayas marcado como favorito. Esto te permite ver visualizaciones de datos clave inmediatamente después de iniciar sesión.

### Establecer un informe favorito

Si aún no has seleccionado un informe favorito, verás un mensaje de bienvenida que te invita a agregar uno.

1.  Ve al área de [Informes](../reports/index.md).
2.  Abre el informe que deseas ver en tu Panel de control.
3.  En la lista de informes o en la vista del informe, encuentra y haz clic en la acción **Agregar a favoritos** (normalmente representada por un icono de estrella).
4.  Actualiza o regresa a tu Panel de control. El informe seleccionado ahora se mostrará.

### Cambiar o eliminar un favorito

Para cambiar tu informe favorito, simplemente agrega un informe diferente a tus favoritos. El nuevo informe reemplazará al anterior en tu Panel de control. Para limpiar el Panel de control, puedes eliminar la configuración del informe favorito del almacenamiento local de tu navegador o contactar a tu administrador del sistema.

## Tour guiado

La primera vez que accedes a Dino, un tour guiado puede iniciarse automáticamente desde el Panel de control para presentarte las principales funciones de la aplicación.

*   Puedes seguir las indicaciones en pantalla para aprender sobre la navegación y las acciones clave.
*   Si omites o finalizas el tour, puedes reiniciarlo en cualquier momento desde el menú de ayuda en el [Área de usuario](../interface/index.md#user-area).

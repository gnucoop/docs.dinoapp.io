# Metriche

Le metriche sono le categorie di dati di riferimento utilizzate in Dino per classificare, organizzare e filtrare i dati raccolti. Quando si compila un modulo o si genera un report, si selezionano valori da queste categorie — ad esempio, scegliendo un progetto, una località o un'organizzazione.

L'area Metriche è il punto in cui si gestiscono gli elenchi di valori disponibili per ciascuna categoria. Funge da hub centrale per tutti i dati di riferimento.

## Tipi di metrica

La pagina principale mostra i tipi di metrica attivi nella tua installazione di Dino. Ogni tipo è visualizzato come una scheda con un'icona e un'etichetta. Fai clic su qualsiasi scheda per aprire la relativa pagina di gestione.

A seconda della configurazione del sistema, potrebbero essere disponibili alcuni o tutti i seguenti tipi di metrica:

| Tipo di metrica | Descrizione |
|---|---|
| **Aree tematiche** | Aree di lavoro o raggruppamenti tematici per le tue attività. |
| **Casi** | Casi individuali, persone o beneficiari monitorati attraverso le compilazioni dei moduli. |
| **Località** | Località geografiche in cui i dati vengono raccolti o si svolgono le attività. |
| **Progetti** | Progetti a cui sono collegati le compilazioni dei moduli e i report. |
| **Organizzazioni** | Organizzazioni coinvolte o responsabili delle attività. |

!!! tip "Accedere alle metriche"
    Puoi navigare nell'area Metriche facendo clic su **Metriche** nel menu principale dell'applicazione.

---

## Cosa puoi fare

Dalla pagina principale delle Metriche puoi:

1.  **Visualizzare tutti i tipi di metrica attivi** disponibili per i tuoi dati.
2.  **Navigare verso un tipo di metrica specifico** facendo clic sulla sua scheda. Questo ti porta a una pagina dedicata dove puoi gestire l'elenco dei valori per quel tipo (ad esempio, aggiungere una nuova località o modificare il nome di un progetto).
3.  **Utilizzare le breadcrumb** nella parte superiore della pagina per tracciare il percorso di navigazione all'interno della sezione Metriche.

Per istruzioni dettagliate sull'aggiunta, modifica o eliminazione di valori all'interno di un tipo di metrica specifico, consulta [Gestire le metriche](managing-metrics.md).

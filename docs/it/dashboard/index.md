# Dashboard

La Dashboard è la prima schermata che vedi dopo aver effettuato l'accesso. Funge da hub centrale per navigare tra le funzionalità di Dino. A seconda della configurazione del sistema, apparirà in uno dei due layout: una **Dashboard Menu** o una **Dashboard Report**.

![Dashboard](../../imgs/dashboard/index.png)

---

## Dashboard Menu

In questo layout, la Dashboard presenta una griglia di schede di navigazione. Ogni scheda fornisce un accesso rapido a un'area principale dell'applicazione che hai il permesso di utilizzare.

Tipicamente vedrai le seguenti schede:

*   **Moduli**: Vai all'area [Moduli](../forms/index.md) per creare nuovi schemi di modulo, raccogliere dati e rivedere le compilazioni.
*   **Report**: Vai all'area [Report](../reports/index.md) per creare, visualizzare e gestire report basati sui dati raccolti.
*   **Metriche**: Vai all'area [Metriche](../metrics/index.md) per gestire dati di riferimento come progetti, luoghi e organizzazioni.
    !!! warning "Visibilità"
        La scheda Metriche è nascosta se il tuo account utente ha solo permessi da ospite.
*   **Utenti**: Vai all'area [Utenti](../users/index.md) per gestire account utente e gruppi.
    !!! tip "Accesso amministratore"
        La scheda Utenti è visibile solo agli utenti con privilegi di amministratore.

Per navigare, fai semplicemente clic sulla scheda dell'area che desideri raggiungere.

## Dashboard Report

In questo layout, la tua Dashboard è personalizzata per visualizzare un singolo report che hai contrassegnato come preferito. Questo ti permette di vedere immediatamente le visualizzazioni dei dati chiave dopo l'accesso.

### Impostare un Report Preferito

Se non hai ancora selezionato un report preferito, vedrai un messaggio di benvenuto che ti invita ad aggiungerne uno.

1.  Vai all'area [Report](../reports/index.md).
2.  Apri il report che desideri vedere sulla tua Dashboard.
3.  Nell'elenco dei report o nella visualizzazione del report, trova e fai clic sull'azione **Aggiungi ai preferiti** (solitamente rappresentata da un'icona a forma di stella).
4.  Aggiorna o torna alla tua Dashboard. Il report selezionato verrà ora visualizzato.

### Cambiare o Rimuovere un Preferito

Per cambiare il tuo report preferito, aggiungi semplicemente un report diverso ai tuoi preferiti. Il nuovo report sostituirà quello precedente sulla tua Dashboard. Per svuotare la Dashboard, puoi rimuovere l'impostazione del report preferito dalla memoria locale del tuo browser o contattare l'amministratore di sistema.

## Tour Guidato

La prima volta che accedi a Dino, potrebbe partire automaticamente un tour guidato dalla Dashboard per presentarti le funzionalità principali dell'applicazione.

*   Puoi seguire le istruzioni a schermo per conoscere la navigazione e le azioni chiave.
*   Se salti o completi il tour, puoi riavviarlo in qualsiasi momento dal menu di aiuto nell'[Area Utente](../interface/index.md#user-area).

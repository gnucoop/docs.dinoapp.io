# Chiacchierare con i Dati (AI)

La pagina Chat Dati ti permette di porre domande in linguaggio semplice sulle risposte raccolte per un modulo. L'assistente AI analizza i dati e restituisce risposte in formato testo, tabelle o grafici.

![Chiacchierare con i Dati (AI)](../../imgs/forms/data-chat.png)

!!! note "Funzionalità opzionale"
    Questa funzionalità richiede una chiave API DataChat e potrebbe non essere disponibile nella tua installazione. Se non vedi questa opzione, contatta il tuo amministratore.

---

## Accettare i Termini di Utilizzo

La prima volta che utilizzi Chat Dati, ti verrà mostrato un documento con i termini di utilizzo. Leggilo e clicca **Accetta** per continuare.

La tua accettazione viene salvata sul tuo dispositivo, quindi non ti verrà richiesto di nuovo nelle visite future.

---

## Inserire la Tua Chiave API

Dopo aver accettato i termini, ti verrà chiesto di inserire la tua **chiave API DataChat**.

1. Digita la tua chiave API nel campo.
2. Premi **Invio** o clicca il **pulsante freccia** per inviarla.

Il sistema verificherà la chiave. Se è valida, verrà caricata l'interfaccia di chat. La tua chiave viene salvata sul tuo dispositivo, quindi non dovrai reinserirla la prossima volta.

!!! warning "Chiave API non valida"
    Se la chiave viene rifiutata, controlla di averla inserita correttamente. Un messaggio di errore apparirà sotto il campo se la chiave non viene riconosciuta.

!!! warning "Servizio non disponibile"
    Se appare il messaggio *"DINO-AI non risponde al momento"*, il servizio potrebbe essere temporaneamente non disponibile. Riprova più tardi.

---

## Porre Domande

Una volta che la chat è pronta, vedrai un'anteprima dei tuoi dati e un campo di testo in fondo allo schermo.

1. Clicca all'interno del campo di testo e digita la tua domanda (ad esempio: *"Quante risposte sono state inviate il mese scorso?"* o *"Mostrami un grafico delle risposte per località"*).
2. Premi **Invio** per inviare.
3. L'assistente elaborerà la tua domanda e visualizzerà la risposta nella cronologia della chat sopra. Le risposte possono apparire come testo, tabelle o grafici.

Puoi continuare la conversazione ponendo domande di approfondimento. Se l'assistente suggerisce domande di approfondimento, puoi cliccarci sopra per inviarle direttamente.

---

## Fornire Feedback

Ogni risposta include un pulsante pollice in su e pollice in giù. Usali per valutare la qualità della risposta. Il tuo feedback aiuta a migliorare l'assistente nel tempo.

---

## Risoluzione dei Problemi

### "Non sono stati trovati moduli"

!!! warning
    Non ci sono ancora risposte associate a questo modulo. Aggiungi alcune voci utilizzando l'elenco delle risposte prima di usare Chat Dati.

### "Spiacente, non hai abbastanza Crediti!"

!!! warning
    Il tuo account ha esaurito i crediti DataChat. Potrebbe apparire un link per acquistare crediti aggiuntivi. Se hai bisogno di aiuto, contatta il tuo amministratore.

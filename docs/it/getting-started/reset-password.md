# Reimpostare la Password

La pagina Reimposta Password ti permette di scegliere una nuova password dopo aver richiesto un link di reimpostazione. Arrivi qui cliccando sul link inviato al tuo indirizzo email — lo stesso link che hai richiesto dall'opzione **"Password dimenticata?"** nella pagina di accesso.

![Reimpostare la Password](../imgs/getting-started/reset-password.png)

!!! note
    Se non hai ancora richiesto un link di reimpostazione, vai prima alla [pagina di accesso](login.md) e clicca **"Password dimenticata?"**.

---

## Impostare una Nuova Password

1.  Apri il link di reimpostazione password dalla tua email. Questo ti porterà direttamente alla pagina Reimposta Password.
2.  Inserisci la tua **nuova password** nel primo campo. La tua password deve essere lunga almeno 9 caratteri.
3.  Reinserisci la stessa password nel campo **Conferma Nuova Password** per assicurarti che corrisponda.
4.  Clicca il **pulsante con la freccia** per confermare la tua nuova password.

Se la password viene accettata, apparirà un messaggio di conferma nella parte superiore dello schermo e verrai reindirizzato automaticamente alla pagina di accesso dopo alcuni secondi. Potrai quindi accedere con la tua nuova password.

---

## Risoluzione dei Problemi

### "Il tuo ticket di reimpostazione password non è valido o è scaduto. Riprova."

!!! warning
    Il link che hai usato non è più valido. I link di reimpostazione scadono dopo poco tempo per motivi di sicurezza. Torna alla [pagina di accesso](login.md), clicca di nuovo **"Password dimenticata?"** e richiedi un nuovo link.

### "Password errata"

!!! warning
    La password non può essere salvata. Assicurati che entrambi i campi contengano esattamente la stessa password e che sia lunga almeno 9 caratteri, poi riprova. Se il problema persiste, richiedi un nuovo link di reimpostazione dalla pagina di accesso.

### La pagina non reindirizza dopo una reimpostazione riuscita

!!! tip
    Se appare il messaggio di conferma ma la pagina non reindirizza automaticamente alla pagina di accesso, navigaci manualmente. La tua password è già stata cambiata con successo.

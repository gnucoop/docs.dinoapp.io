# Tableau de bord

Le Tableau de bord est le premier écran que vous voyez après vous être connecté. Il sert de centre névralgique pour naviguer dans les fonctionnalités de Dino. Selon la configuration de votre système, il apparaîtra dans l'un des deux agencements suivants : un **Tableau de bord Menu** ou un **Tableau de bord Rapport**.

![Tableau de bord](../../imgs/dashboard/index.png)

---

## Tableau de bord Menu

Dans cet agencement, le Tableau de bord présente une grille de cartes de navigation. Chaque carte offre un accès rapide à un domaine principal de l'application que vous avez l'autorisation d'utiliser.

Vous verrez généralement les cartes suivantes :

*   **Formulaires** : Accédez à la zone [Formulaires](../forms/index.md) pour créer de nouveaux schémas de formulaire, collecter des données et examiner les soumissions.
*   **Rapports** : Accédez à la zone [Rapports](../reports/index.md) pour créer, visualiser et gérer des rapports basés sur vos données collectées.
*   **Métriques** : Accédez à la zone [Métriques](../metrics/index.md) pour gérer les données de référence comme les projets, les lieux et les organisations.
    !!! warning "Visibilité"
        La carte Métriques est masquée si votre compte utilisateur dispose uniquement des permissions invité.
*   **Utilisateurs** : Accédez à la zone [Utilisateurs](../users/index.md) pour gérer les comptes utilisateurs et les groupes.
    !!! tip "Accès administrateur"
        La carte Utilisateurs n'est visible que par les utilisateurs disposant de privilèges d'administrateur.

Pour naviguer, cliquez simplement sur la carte correspondant à la zone que vous souhaitez accéder.

## Tableau de bord Rapport

Dans cet agencement, votre Tableau de bord est personnalisé pour afficher un seul rapport que vous avez marqué comme favori. Cela vous permet de visualiser immédiatement des visualisations de données clés dès la connexion.

### Définir un rapport favori

Si vous n'avez pas encore sélectionné de rapport favori, vous verrez un message de bienvenue vous invitant à en ajouter un.

1.  Rendez-vous dans la zone [Rapports](../reports/index.md).
2.  Ouvrez le rapport que vous souhaitez voir sur votre Tableau de bord.
3.  Dans la liste des rapports ou la vue du rapport, trouvez et cliquez sur l'action **Ajouter aux favoris** (généralement représentée par une icône en forme d'étoile).
4.  Actualisez ou retournez sur votre Tableau de bord. Le rapport sélectionné s'affichera désormais.

### Changer ou supprimer un favori

Pour changer votre rapport favori, ajoutez simplement un rapport différent à vos favoris. Le nouveau rapport remplacera l'ancien sur votre Tableau de bord. Pour effacer le Tableau de bord, vous pouvez supprimer le paramètre de rapport favori du stockage local de votre navigateur, ou contacter votre administrateur système.

## Visite guidée

La première fois que vous accédez à Dino, une visite guidée peut démarrer automatiquement depuis le Tableau de bord pour vous présenter les principales fonctionnalités de l'application.

*   Vous pouvez suivre les instructions à l'écran pour découvrir la navigation et les actions clés.
*   Si vous ignorez ou terminez la visite, vous pouvez la redémarrer à tout moment depuis le menu d'aide dans la [Zone Utilisateur](../interface/index.md#user-area).

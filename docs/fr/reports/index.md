# Rapports

La zone Rapports est votre centre d'accès à tous les schémas de rapport disponibles. Un schéma de rapport définit la structure et le contenu d'un rapport pouvant être généré à partir de vos données collectées. À partir d'ici, vous pouvez parcourir les schémas et accéder aux rapports qui ont déjà été créés.

![Rapports](../../imgs/reports/index.png)

---

## Parcourir les schémas de rapport

Lorsque vous ouvrez la page Rapports, vous voyez une liste de tous les schémas de rapport auxquels vous avez le droit d'accéder. La liste est triée par ordre alphabétique selon le libellé du schéma.

Pour trouver un schéma spécifique :
1.  Utilisez la barre de recherche en haut de la page.
2.  Tapez une partie du nom ou du libellé du schéma.
3.  La liste se filtrera en temps réel pour n'afficher que les schémas correspondants.

Pour voir les rapports d'un schéma spécifique, cliquez sur sa carte dans la liste.

!!! tip "Pour commencer"
    Si vous voyez le message « Il n'y a aucun rapport », cela signifie qu'aucun schéma de rapport n'a encore été créé ou partagé avec vous. Contactez votre administrateur Dino pour commencer.

---

## Ce que vous pouvez faire

À partir de la page principale Rapports, vous pouvez naviguer pour effectuer ces actions clés :

*   **[Créer un rapport](creating-a-report.md)** — Générez un nouveau document de rapport en sélectionnant un schéma, en choisissant vos métriques et en définissant une plage de dates.
*   **[Voir et gérer les rapports](viewing-reports.md)** — Parcourez, visualisez, exportez, ajoutez en favori ou supprimez les rapports qui ont déjà été générés.
*   **[Gérer les schémas de rapport](managing-templates.md)** — Créez de nouveaux schémas de rapport ou modifiez ceux existants pour définir les données qui apparaissent dans les rapports. Cela nécessite généralement des autorisations d'administrateur.

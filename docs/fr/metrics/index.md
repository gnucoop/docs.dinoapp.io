# Métriques

Les métriques sont les catégories de données de référence utilisées dans Dino pour classer, organiser et filtrer vos données collectées. Lorsque vous remplissez un formulaire ou générez un rapport, vous sélectionnez des valeurs dans ces catégories — par exemple, en choisissant un projet, un lieu ou une organisation.

La zone Métriques est l'endroit où vous gérez les listes de valeurs disponibles pour chaque catégorie. Elle sert de centre névralgique pour toutes vos données de référence.

![Métriques](../../imgs/metrics/index.png)

---

## Types de métriques

La page principale affiche les types de métriques actifs dans votre installation Dino. Chaque type est présenté sous forme de carte avec une icône et un libellé. Cliquez sur n'importe quelle carte pour ouvrir sa page de gestion.

Selon la configuration de votre système, certains ou tous les types de métriques suivants peuvent être disponibles :

| Type de métrique | Description |
|---|---|
| **Domaines thématiques** | Domaines de travail ou regroupements thématiques pour vos activités. |
| **Cas** | Cas individuels, personnes ou bénéficiaires suivis à travers les soumissions de formulaires. |
| **Lieux** | Emplacements géographiques où les données sont collectées ou où les activités se déroulent. |
| **Projets** | Projets auxquels les soumissions de formulaires et les rapports sont liés. |
| **Organisations** | Organisations impliquées dans ou responsables des activités. |

!!! tip "Accéder aux Métriques"
    Vous pouvez accéder à la zone Métriques en cliquant sur **Métriques** dans le menu principal de l'application.

---

## Ce que vous pouvez faire

Depuis la page principale des Métriques, vous pouvez :

1.  **Voir tous les types de métriques actifs** disponibles pour vos données.
2.  **Naviguer vers un type de métrique spécifique** en cliquant sur sa carte. Cela vous amène à une page dédiée où vous pouvez gérer la liste des valeurs pour ce type (par exemple, ajouter un nouveau lieu ou modifier un nom de projet).
3.  **Utiliser le fil d'Ariane** en haut de la page pour suivre votre chemin de navigation dans la section Métriques.

Pour des instructions détaillées sur l'ajout, la modification ou la suppression de valeurs au sein d'un type de métrique spécifique, consultez [Gérer les métriques](managing-metrics.md).

# Agrégation

La page d'agrégation offre une vue unifiée de toutes les soumissions de formulaires au sein de votre organisation. Au lieu de consulter les soumissions pour un seul schéma de formulaire à la fois, vous pouvez voir les entrées de plusieurs schémas regroupées dans une seule liste, filtrable.

![Agrégation](../../imgs/aggregation/index.png)

## Parcourir la liste

La liste affiche toutes les soumissions que vous avez l'autorisation de consulter. Chaque ligne représente une soumission unique. Les colonnes suivantes sont affichées par défaut :

*   **Schéma de formulaire** : Le nom du schéma de formulaire auquel appartient cette soumission.
*   **Statut** : Le statut actuel du flux de travail de la soumission.
*   Des colonnes supplémentaires pour des métriques comme **Projet**, **Localisation** ou **Organisation** peuvent également apparaître, selon la configuration de votre système.

Vous pouvez cliquer sur n'importe quelle ligne pour la sélectionner ou l'étendre pour voir plus de détails.

## Filtrer la liste

Une barre de filtres est située au-dessus de la liste. Utilisez-la pour affiner les soumissions affichées. Vous pouvez filtrer par :

*   Projet
*   Localisation
*   Zone
*   Cas
*   Code de cas
*   Organisation
*   Statut du formulaire
*   Utilisateur

!!! tip "Conseils de filtrage"
    La barre de filtres de l'agrégation est conçue pour un filtrage rapide et transversal aux schémas. Pour des fonctionnalités avancées comme des préréglages de filtres enregistrés ou l'exportation de données, naviguez vers la liste des soumissions pour un schéma de formulaire spécifique.

## Actions sur les lignes

Lorsque vous survolez une ligne, un ensemble d'icônes d'action apparaît à droite. Les actions disponibles pour une soumission spécifique dépendent de vos autorisations pour son schéma de formulaire.

*   **Voir** (![icône œil]()) : Ouvrir la soumission en vue seule.
*   **Modifier** (![icône crayon]()) : Ouvrir la soumission pour la modifier.
*   **Imprimer** (![icône imprimante]()) : Générer et ouvrir une version PDF de la soumission. Une confirmation vous sera demandée avant la création du PDF.
*   **Supprimer** (![icône corbeille]()) : Supprimer définitivement la soumission. Cette action nécessitera une confirmation.

## Créer une nouvelle soumission

Vous pouvez démarrer une nouvelle soumission de formulaire directement depuis la page d'agrégation.

1.  Cliquez sur le bouton **+** (Ajouter) dans le coin inférieur droit de l'écran.
2.  Une boîte de dialogue s'ouvre, affichant une liste des schémas de formulaire pour lesquels vous avez l'autorisation de créer des soumissions.
3.  Sélectionnez le schéma de formulaire souhaité dans la liste.
4.  Cliquez sur **Créer un formulaire**. Vous serez redirigé vers le formulaire pour commencer à le remplir.

# Consultation et gestion des soumissions

Cette page affiche toutes les soumissions pour un schéma de formulaire spécifique. Vous pouvez parcourir, rechercher, consulter les détails, modifier, exporter et gérer des entrées individuelles à partir de cette liste.

---

## Parcourir la liste

La liste affiche toutes les soumissions pour le formulaire sélectionné. Le nombre total d'entrées correspondant à vos filtres actuels est indiqué en haut.

Utilisez les **contrôles de pagination** pour naviguer entre les pages, ou accéder directement à la première ou dernière page.

Vous pouvez également modifier les **colonnes** visibles en cliquant sur le **bouton de sélection de colonnes** (icône de grille) dans l'en-tête du tableau.

---

## Rechercher et filtrer

Une barre de filtres au-dessus de la liste vous permet de réduire le nombre d'entrées affichées :

- **Recherche par mot-clé** — saisissez un mot pour filtrer les entrées dont les champs contiennent ce texte.
- **Date de début / Date de fin** — affiche uniquement les entrées créées dans une plage de dates.
- **Filtres de métriques** — filtrez par projet, localisation, zone, dossier, organisation ou autres données liées.
- **Filtre de statut** — affiche uniquement les entrées ayant un statut de workflow spécifique.
- **Filtre d'utilisateur** — affiche uniquement les entrées créées par un utilisateur ou groupe d'utilisateurs spécifique.
- **Filtres enregistrés (préréglages)** — enregistrez votre combinaison de filtres actuelle comme préréglage pour un accès rapide ultérieur. Utilisez le menu des préréglages dans la barre de filtres pour enregistrer, charger ou supprimer des préréglages.

Pour supprimer un filtre, cliquez sur le **×** à l'intérieur de ce champ.

---

## Actions sur une ligne

Chaque ligne dispose d'un ensemble d'icônes d'action sur la droite. Les actions disponibles dépendent de vos autorisations :

- **Consulter** (icône d'œil) — ouvrir l'entrée en mode lecture seule.
- **Modifier** (icône de crayon) — ouvrir l'entrée pour modification.
- **Imprimer** (icône d'imprimante) — générer une version PDF de l'entrée.
- **Exporter en Word** (icône de document) — télécharger l'entrée sous forme de document Word.
- **Dupliquer** (icône de copie) — créer une nouvelle entrée brouillon pré-remplie avec les données de cette entrée.
- **Supprimer** (icône de poubelle) — supprimer définitivement l'entrée. Une demande de confirmation apparaîtra.
- **Imprimer un badge** (icône de badge) — générer un document de badge pour cette entrée (disponible si le formulaire est lié à un dossier).
- **Voir le journal** (icône d'historique) — consulter l'historique des modifications apportées à cette entrée.

!!! note "Actions basées sur les autorisations"
    Toutes les actions ne sont pas visibles par chaque utilisateur. Les actions affichées dépendent des autorisations qui vous sont attribuées. Certaines actions, comme la modification ou la suppression, peuvent n'être disponibles que si le statut de l'entrée le permet.

---

## Actions groupées

Pour agir sur plusieurs entrées simultanément :

1. Sélectionnez les entrées que vous souhaitez gérer en cochant la case sur leurs lignes.
2. Utilisez les **boutons d'action groupée** qui apparaissent dans la barre d'outils pour supprimer ou modifier toutes les entrées sélectionnées en une seule fois.

!!! warning "Restrictions sur la modification groupée"
    La modification groupée n'est disponible que si toutes les entrées sélectionnées ont un statut autorisant la modification.

---

## Ajouter une nouvelle entrée

Cliquez sur le **bouton +** (bouton circulaire flottant en bas à droite de la page) pour ouvrir un formulaire vierge et enregistrer une nouvelle soumission.

---

## Importer des entrées

Cliquez sur le **bouton d'importation** (icône de téléchargement depuis le cloud, bouton flottant en bas à droite) pour importer en masse des entrées à partir d'un fichier.

---

## Exporter la liste

Cliquez sur le **bouton Exporter** (icône de téléchargement) dans la barre de filtres pour télécharger la liste actuelle des entrées sous forme de fichier.

---

## Visualiser les données sur une carte

Si les soumissions incluent des données géographiques, cliquez sur le **bouton Carte** dans la barre de filtres pour ouvrir la [Vue Carte](map-view.md) pour ce formulaire.

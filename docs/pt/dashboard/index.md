# Painel de Controle

O Painel de Controle é a primeira tela que você vê após fazer login. Ele serve como seu hub central para navegar pelos recursos do Dino. Dependendo da configuração do seu sistema, ele aparecerá em um de dois layouts: um **Painel de Controle de Menu** ou um **Painel de Controle de Relatórios**.

![Painel de Controle](../../imgs/dashboard/index.png)

---

## Painel de Controle de Menu

Neste layout, o Painel de Controle apresenta uma grade de cartões de navegação. Cada cartão fornece acesso rápido a uma área principal do aplicativo que você tem permissão para usar.

Você normalmente verá os seguintes cartões:

*   **Formulários**: Navegue até a área de [Formulários](../forms/index.md) para criar novos esquemas de formulário, coletar dados e revisar envios.
*   **Relatórios**: Navegue até a área de [Relatórios](../reports/index.md) para criar, visualizar e gerenciar relatórios com base nos dados coletados.
*   **Métricas**: Navegue até a área de [Métricas](../metrics/index.md) para gerenciar dados de referência como projetos, locais e organizações.
    !!! warning "Visibilidade"
        O cartão Métricas fica oculto se sua conta de usuário tiver permissões apenas de convidado.
*   **Usuários**: Navegue até a área de [Usuários](../users/index.md) para gerenciar contas de usuário e grupos.
    !!! tip "Acesso de Administrador"
        O cartão Usuários é visível apenas para usuários com privilégios de administrador.

Para navegar, basta clicar no cartão da área que deseja acessar.

## Painel de Controle de Relatórios

Neste layout, seu Painel de Controle é personalizado para exibir um único relatório que você marcou como favorito. Isso permite que você visualize visualizações de dados-chave imediatamente após o login.

### Definindo um Relatório Favorito

Se você ainda não selecionou um relatório favorito, verá uma mensagem de boas-vindas solicitando que você adicione um.

1.  Vá para a área de [Relatórios](../reports/index.md).
2.  Abra o relatório que deseja ver no seu Painel de Controle.
3.  Na lista de relatórios ou na visualização do relatório, encontre e clique na ação **Adicionar aos favoritos** (geralmente representada por um ícone de estrela).
4.  Atualize ou retorne ao seu Painel de Controle. O relatório selecionado agora será exibido.

### Alterando ou Removendo um Favorito

Para alterar seu relatório favorito, basta adicionar um relatório diferente aos seus favoritos. O novo relatório substituirá o antigo no seu Painel de Controle. Para limpar o Painel de Controle, você pode remover a configuração do relatório favorito do armazenamento local do seu navegador ou entrar em contato com o administrador do sistema.

## Tour Guiado

Na primeira vez que você acessar o Dino, um tour guiado pode iniciar automaticamente a partir do Painel de Controle para apresentar os principais recursos do aplicativo.

*   Você pode seguir as instruções na tela para aprender sobre navegação e ações principais.
*   Se você pular ou concluir o tour, pode reiniciá-lo a qualquer momento no menu de ajuda na [Área do Usuário](../interface/index.md#user-area).
